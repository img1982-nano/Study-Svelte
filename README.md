# Aziquiz(改良版)
## 概要 / Overview
Svelte学習の際に作成した、Aziquizの改良版です。  
データベースに様々な問題集(予定)があり、ユーザーが自由に問題集を追加することが可能です。(予定)  
AIを使用して、ユーザーの回答と問題分から解説を生成します。  
ポイント機能で、勉強の成果をデータ化(予定)  
(ほぼほぼ未実装)
